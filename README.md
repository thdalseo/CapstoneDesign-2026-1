# Bridge - 유학생·내국인 학생 문화 교류 플랫폼

<br>

## 개요

한국 대학 내 외국인 유학생 수는 꾸준히 증가하고 있으나, 유학생과 내국인 학생 간의 실질적인 교류는 여전히 부족한 실정이다. 대부분의 대학은 이를 해결하기 위해 버디 프로그램, 튜터링 등 오프라인 기반의 교류 프로그램을 운영하고 있다.

그러나 기존 프로그램은 다음과 같은 한계를 가진다.

- 선발 절차가 복잡하여 교류를 원하는 학생이 시작 단계에서 포기하는 경우가 발생
- 한국 학생의 경우 일정 수준 이상의 외국어 능력을 요구해 참여 가능한 학생 자체가 제한
- 수동 매칭 방식으로 개인 성향이 충분히 반영되지 않아 일회성 만남에 그치는 경우가 많음
- 언어 소통을 지원하는 도구가 없어 외국어에 자신 없는 학생은 실질적인 대화로 이어지기 힘듦

**Bridge**는 학교 메일 인증을 통해 검증된 사용자 간의 신뢰 기반 네트워크를 구축하며, 관심사·학과·성향 등을 반영한 가중치 알고리즘으로 개인화 매칭을 제공한다. 또한 실시간 AI 번역을 통해 외국어 능력과 무관하게 누구나 교류할 수 있는 환경을 실현한다.

<br>

## 목표

학교 메일 인증 기반의 내·외국인 학생 교류 모바일 앱을 개발한다.

**과학·기술적 측면**
사용자의 관심사·학과·성향 데이터를 기반으로 최적의 교류 대상을 추천하는 매칭 알고리즘을 설계·구현한다. 기존 교류 앱이 국적이나 학습 언어 정도의 단순 조건만 제공했던 것과 달리, 보다 세밀한 개인화 매칭을 기술적으로 구현하는 것을 목표로 한다. 아울러 실시간 AI 번역 기능을 채팅에 통합하여 언어 능력과 무관하게 원활한 소통이 가능한 환경을 실현한다.

**사회적 측면**
언어 장벽과 복잡한 절차로 인해 교류 기회를 얻지 못했던 학생들이 보다 쉽게 연결될 수 있는 환경을 조성한다. 유학생의 초기 캠퍼스 적응을 실질적으로 지원하고, 내·외국인 학생 간 지속적인 교류를 통해 상호 문화 이해를 증진함으로써 대학 내 다양성과 포용적 공동체 형성에 기여한다.

## 핵심 기능

| 기능 | 설명 |
|------|------|
| 학교 메일 인증 | 학교 이메일 기반 신뢰 네트워크 구축 |
| 스마트 매칭 | 관심사·학과·성향 기반 가중치 알고리즘으로 개인화 매칭 |
| 실시간 번역 채팅 | AI 번역 + 아이스브레이킹 질문 가이드 제공 |
| 헬피-헬퍼 보드 | 행정·학업·생활 등 카테고리별 도움 요청 및 수락 시스템 |
| 로컬 가이드 | 유학생 맞춤형 학교 주변 정보 공유 |
| 미션/챌린지 | 지속적인 교류를 유도하는 주간 미션 시스템 |

## 시작하기

### 요구사항
- Python 3.11 이상
- Flutter SDK
- MySQL

### 백엔드 실행

```bash
cd backend
uv sync
uv run python -m uvicorn main:app --reload
```

### 환경변수 설정

```bash
cp .env.example .env
# .env 파일에 필요한 값 입력
```
